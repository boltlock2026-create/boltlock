   - Renovação automática
   - Cancelamento: 30 dias antecedência

   - Incluídas: 1.000/mês (Plano Base)
   - Excedentes: R$ 0,50/consulta
   - Créditos não utilizados expiram em 90 dias

4. CUSTOS DE API (PASS-THROUGH)
   - Cliente paga custos reais de APIs utilizadas
   - Você atua como intermediária (sem margem, sem subsídio)
   - Faturamento separado ou incluído a critério comercial

5. SLA (Service Level Agreement)
   - Uptime: 99.9%
   - Suporte: 8h-18h (seg-sex)
   - Response time: 4h

6. DADOS
   - Dados do cliente são propriedade do cliente
   - Backup diário (cliente pode exportar)
   - Retenção de logs: 90 dias
   - Exclusão: 30 dias após cancelamento

7. CANCELAMENTO
   - Sem multa
   - 30 dias de aviso prévio
   - Dados exportáveis
   - Reembolso: proporcional ao mês
